StarDict
========

Include:

1. stardict

2. sdcv

Install
-------

Install stardict and sdcv::

    sudo apt-get install stardict sdcv
    sudo yum install stardict sdcv

Configuration::

    tar -jxvf ~/pydeveloper/data/dict/*.tar.bz2 -C /usr/share/stardict/dic/

How to use
----------

Use it in vim::

    Ctrl + d

Use sdcv in command line::

    sdcv hello

Use stardict with GUI::

    stardict hello
